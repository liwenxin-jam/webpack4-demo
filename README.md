# webpack4-study-demo

#### Webpack4学习配置
> 基于webpack4的个人学习项目，关于module、loader、plugin在代码上基本上都有注释。这里就不一一拓展开讲，个人觉得还是直接看src源码注释来得实际。有兴趣的童鞋可以给个star，fork下学习。

* src文件夹 单入口模式 webpack.config.js
* multiple-src文件夹 多入口模式 webpack.mul.config.js
